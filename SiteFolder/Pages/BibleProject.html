Description

The purpose of this project is to replicate and enhance some features of an existing Bible app: Bible Browser App.

I decided to build my own version because I thought it would be a fun project, and there were features I wanted to add that I found relatively easy to implement.

After working on this project, I realized that their search engine and overall product are actually very good :)

I plan to continue improving this project over time.

These are a few simple features I felt were missing from the previously mentioned app:

  • Improved search engine
  • Dark mode
  • Favorites section
  • Synchronization of favorites across devices
  • Bionic reading (read more here)
