Description

The objective of this project is trying to copy and add some features to an already existing bible appbible browser app.

My motivation to do this, was because there were some features I would like the previous app to have and it didn't so I made my own to my liking

After creating this project I notice that after all their search engine and product overall are actually very good :)

I will try to make this project evolve with time.

Some simple features I wished the previously mentioned app had are the following:

  • Better search engine
  • Dark mode
  • Favorites section
  • Synchronization between computers of favorite sction
  • Bionic reading

This is a still in development project.
