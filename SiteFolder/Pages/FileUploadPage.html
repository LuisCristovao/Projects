File Upload Server Description:


This project, as the title says is server that allows a user to upload files and store them. This project was made in Bottle: Python Web Framework, meaning in Python. There is a live demo of this project on heroku but as I am using the free server heroku gives to users, when the server is not used for a long period of time, it restarts the server Deleting all Files Uploaded! So it is possible to use this as a upload server for a short amount of time.
