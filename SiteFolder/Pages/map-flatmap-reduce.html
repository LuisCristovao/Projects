ForEach

If you don't know anything about it, here it is a little explanation. Imagine you have the following function:

function foreach(some_array,some_function_action){ for (i=0;i<some_array.length;i++){ var element=some_array[i] some_function_action(element) } }

This function receives an array and a function as input. The input function will receive each element of the array and do something with it. This function is essentially what foreach function does.

Let us see in action:

var arr=[1,2,3,4,5] foreach(arr,(element)=>{console.log(`Number: ${element}`)}) //This will print all numbers of array like this: //Number: 1 //Number: 2 //Number: 3 // ...

To test this example copy both codes (this and the previous one by order!!) to the browser console.

forEach Usage

Example 1

var arr=[1,2,3,4,5] arr.forEach(element=>{console.log(`Number: ${element}`)})

Map

The map function is the same ideia than foreach but it returns an array in the end. Check this example.

function map(some_array,some_function_action){ var out_array=[] for (i=0;i<some_array.length;i++){ var element=some_array[i] out_array.push(some_function_action(element)) } return out_array }

The only difference here is that in this case we are returning an array with the changes made by some_function_action to each element of input array(some_array)

Now lets us see an example:

var arr=[1,2,3,4,5] var new_array=map(arr,(element)=>{element*2}) new_array.forEach(el=>console.log(el))
