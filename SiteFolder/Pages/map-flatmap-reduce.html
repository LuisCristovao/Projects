ForEach

Explanation

If you don't know anything about it, here it is a little explanation. Imagine you have the following function:

function foreach(some_array,some_function_action){ for (i=0;i<some_array.length;i++){ var element=some_array[i] some_function_action(element) } }

This function receives an array and a function as input. The input function will receive each element of the array and do something with it. This function is essentially what foreach function does.

Let us see in action:

var arr=[1,2,3,4,5] foreach(arr,(element)=>{console.log(`Number: ${element}`)}) //This will print all numbers of array like this: //Number: 1 //Number: 2 //Number: 3 // ...

To test this example copy both codes (this and the previous one by order!!) to the browser console.

forEach Usage

Example 1

Map

Map Explanation

The map function is the same ideia than foreach but it returns an array in the end. Check this example.

function map(some_array,some_function_action){ var out_array=[] for (i=0;i<some_array.length;i++){ var element=some_array[i] out_array.push(some_function_action(element)) } return out_array }

The only difference here is that in this case we are returning an array with the changes made by some_function_action to each element of input array(some_array)

Now lets us see an example:

var arr=[1,2,3,4,5] var new_array=map(arr,(element)=>{return element*2}) new_array.forEach(el=>console.log(el))

To test this example copy both codes (this and the previous one by order!!) to the browser console.

Map Usage

Example 1

var arr=[1,2,3,4,5] var new_array=arr.map(element=>{return element*2}) new_array.forEach(el=>console.log(el))

Reduce

Reduce Explanation

Reduce function is a little bit different, but its purpose is to aggregate arrays data into one value.

Check this example where we implement the reduce function:

//functions---------- function reduce(some_array,some_function_action,initial_value){ var accumulator=initial_value for (i=0;i<some_array.length;i++){ var next_val=some_array[i] accumulator=some_function_action(accumulator,next_val) console.log(accumulator) } return accumulator } function sum(a,b){ return a+b } //Main -------- var array=[1,2,3,4,5,6] reduce(array,sum,0)

See this code in action

As you can see what the reduce function is doing in this example is sum all values from the array. The sum, in this case, is the aggregating function. Check other examples down below

Reduce Usage

Sum Example

//Sum example var array=[1,2,3,4,5,6] array.reduce((acc,next)=>{return acc+next},0)//initial value of acc=0 //without initial value it will consider 0 as default

Concate strings with comma ','

var array=[" X "," Y ", "Z"] var reduce_without_initial_value=array.reduce((acc,next)=>{return `${acc.trim()},${next.trim()}`}) console.log(reduce_without_initial_value) var reduce_with_initial_value=array.reduce((acc,next)=>{return `${acc.trim()},${next.trim()}`},"") console.log(reduce_with_initial_value) var other_example=array.reduce((acc,next)=>{return `${acc.trim()},${next.trim()}`}," A ".trim()) console.log(other_example)

Create csv from array with data

//data to csv file string var data=[["id","name","cash"],["1","peter","100"],["2","james","200"]] var data1=data.map(el=>el.reduce((acc,next)=>{return `${acc},${next}`})) console.log(data1) var data2=data1.reduce((acc,next)=>{return `${acc}\n${next}`}) console.log(data2)
