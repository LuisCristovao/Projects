## Intro
**To test shell scripting there is these options (that I know of):**
1. Have a pc with Linux operating system (OP)
2. Use **vagrant** ([link for tutorial here](https://www.vagrantup.com/intro/getting-started/index.html)) or **virtual machine** with a Linux OP
3. Use **git Bash**
4. Search on google keywords: **shell or bash script online**

If you have **Windows (OP)** and want rapidly to test shell scripting you can start with options **4(simple stuff only)** and **3(more powerful)**. But if you want to go more deep in the knowledge of Unix better solutions are **1** and **2**
