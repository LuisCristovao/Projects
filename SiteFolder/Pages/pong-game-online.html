Local Multiplayer Pong Game

I created a classic Pong game with local multiplayer using WebRTC favorite library, PeerJS. Feel free to try the game yourself and play with a friend!

Play Game

Developer Notes:

  • The main game loop was created with the help of AI (Grok3), so there were no major issues here.
  • The multiplayer part, however, was not implemented as I intended, so I had to assist the AI:
    • I used another project, where I built a chat application using PeerJS, as a template for this project.
    • Basically, if both players are on the same network, PeerJS only needs a connection string to establish the connection between them.
    • One issue to consider is implementing multiple connection attempts in case of failure.
    • Another challenge was determining when the connection was successfully established and preventing PeerJS from initiating additional connections with other players afterward.
    • Fixing game bugs, such as an issue where the game speed was increasing exponentially.
  • Although the game runs well for the non-host player when there is a good connection, some lag is inevitable. However, it is still playable.
  • Unfortunately, while PeerJS is a great tool, achieving smooth performance requires further optimizations.
