Basic ML Server Description:

This project, as the title says is a basic machine learning server that allows the user to export the model in the end. The server will accept csv like files, for instance a .txt with the features separeted by a " , " or any othe character. The target class must be in the last right column in order to work!! Also I made this server using Bottle: Python Web Framework

[DISCLAIMER]: This project is really limited! It uses basic machine learning algorithms, and has lots of other limitations, such as, does not do interpolations... just the basic features target stuff. There are more limitations but it is not necessary to display them here.
