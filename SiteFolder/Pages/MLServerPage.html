Basic ML Server Description:

This project, as the title says is a basic machine learning server that allows the user to export the model in the end from the train dataset. The server will accept csv like files, for instance a .txt with the features separeted by a " , " or any othe character. The target class must be in the last right column in order to work!! Also I made this server using Bottle: Python Web Framework

Below there is a live demo, The Source Code actually running in Heroku app and a small video tutorial how to use the application server.
