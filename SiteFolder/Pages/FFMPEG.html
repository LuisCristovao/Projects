FFMPEG Knowledge

Turn image sequence to video

This command works if you are in a directory with images(jpg) with names such as 0.jpg,1.jpg,....

ffmpeg -r 60 -f image2 -i %d.jpg video.avi

Convert video to gif

-r is for the fps

ffmpeg -r 50 -i video.avi -pix_fmt rgb24 out.gif

Change video or image size

Video example

ffmpeg -i video.mp4 -filter:v scale=720:-1 -c:a copy videoLowRes.mp4

Image example change to width:320; height:270

ffmpeg -i in.gif -vf scale=320:270 outLowRes.gif

Video/GIF to image sequence (24 fps)

ffmpeg -i video.mp4 -start_number 0 -r 24 %d.jpg ffmpeg -i video.gif -start_number 0 -r 24 %d.png
