Volley Game Description:

This project is a local multiplayer volley game, made in Unity. It is possible to play up to 4 players locally in chrome browser. The Game has 2 modes "Normal Mode" and "Crazy Mode", the difference is that in Crazy Mode the physics of the game change on each round.
