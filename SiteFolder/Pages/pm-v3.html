Password Manager v3

Goals

  • Manage and Store information about passwords and other sensitive information in a secure way
  • Synchronize information between multiple devices in an efficient way
  • Work offline:
    • For now you need to use a HTML file for this purpose
    • It has more limitations than running online but has access to manage passwords
  • Work on both mobile phones and PCs
  • Has the ability to import or export data (encrypted or decrypted)
  • Code is more scalable
  • Special feature:
    • Using QR scanner library to simplify sync between devices
  • No installation required

Techincal Details

  • Data encrypted with AES-GCM
  • Data Stored using browser IndexDB
  • Data synchronization between devices using webRTC
  • Usage of QRcode scanner to connect devices
