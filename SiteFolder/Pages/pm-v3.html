Password Manager v3

Goals

  • Manage and Store information about passwords and other sensitive information in a secure way
  • Syncronize information between multiples devices in efficient way
  • Work offline:
    • For now you need to use a html file for this purpose
  • Work both in mobile phones and PC's
  • Has the ability of import or export data (encrypted or decrypted)
  • Code more be escalable.
  • Special feature:
    • Using QR scanner lib to sync between devices
  • No installation required

Techincal Details

  • Data encrypted with AES-GCM
  • Data Stored using browser IndexDB
  • Data synchronization between devices using webRTC
  • Usage of QRcode scanner to connect devices
