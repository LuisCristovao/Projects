Sources

This post is just a practical summary of the information on this SITE

The following subtopics will show step by step how to turn your pc into a wifi router even without an internet connection

Check if is possible

Open a command line as Admin and execute the following code:

NETSH WLAN show drivers

Check if the result on your pc is similar

Create wifi hotspot

To create a wifi connection available to others with some name and password execute the following:

NETSH WLAN set hostednetwork mode=allow ssid=<name> key=<password>

Start Wireless hotspot

Execute command:

NETSH WLAN start hostednetwork

Open Control Panel > Network and Internet > Network Connectios to see the newly created wifi connection

Now to see some fun things you can do with this go to the chapter Fun Application Examples

Close wifi hotspot

Execute commands by order

NETSH WLAN stop hostednetwork

If you see in Open Control Panel > Network and Internet > Network Connectios the newly created wifi hotspot is gone.

But to complete closing execute:

netsh wlan set hostednetwork mode=disallow
